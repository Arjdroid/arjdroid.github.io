About The Website

This website was built using Hugo, a powerful and fast open-source engine for building great static HTML websites.

This website is being hosted by GitHub, a fantastic resource enabling open-source content to be distributed accessibly worldwide.

Most of the edits were made in Atom a ‘hackable text editor for the 21st century’ supplemented by a zsh command shell running on Arch Linux in the Alacritty Terminal. The Hugo theme that I am using is Stack.
