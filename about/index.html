About The Website

This website is built using Hugo, a relatively simple, open-source engine for building great static HTML websites.

The Hugo theme that I am using is Stack, by Jimmy Cai

This website is hosted on GitHub (specifically, Github Pages), a fantastic resource enabling open-source content to be distributed accessibly worldwide.

Most of the edits were made in Atom a ‘hackable text editor for the 21st century’ supplemented by a zsh command shell running on Arch Linux in the Alacritty Terminal.

Nowadays I use VSCodium alongside nano as Atom is no longer supported. I have tried getting into emacs but it is quite immense and I have not explored it as much yet.

All the photographs that appear on this website are shot by me, either on an Olympus OMD E-M10 Mark II micro-four-thirds digital mirrorless camera or on my smartphone or are screenshots taken by me, on my devices.
